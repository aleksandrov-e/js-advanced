# JavaScript Advanced

This repository contains exercises and materials from the **JavaScript Advanced** course.

## 📚 Topics Covered

* Functions and Statements
* Arrays and Nested Arrays
* Objects and Composition
* DOM Introduction
* DOM Manipulations and Events
* Advanced Functions
* Unit Testing
* Classes
* Prototypes and Inheritance
* Interactive Applications

## 🎯 Purpose

The goal of this repository is to improve problem-solving skills and gain a deeper understanding of JavaScript, including:

* Working with complex data structures
* Understanding how JavaScript works under the hood
* Manipulating the DOM
* Writing clean and reusable code
* Testing applications

## 🗂️ Structure

Each folder represents a specific topic and contains multiple exercises and solutions.

## 🛠️ Technologies Used

* JavaScript (ES6+)
* HTML & CSS (for DOM tasks)
